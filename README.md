Projeto JavaScript – Interações de Interface

Projeto desenvolvido durante o curso da Origamid com foco na implementação de funcionalidades interativas utilizando JavaScript puro.

O projeto explora diferentes padrões de interação comuns em interfaces modernas:

Navegação por Tabs: alternância dinâmica de conteúdo sem recarregar a página;
Scroll Suave (Smooth Scroll): navegação fluida entre seções da página;
Accordion List: exibição de conteúdo expansível/recolhível;
Linkagem de Imagens com Conteúdo: interação visual conectando elementos da interface;
Modais (Modal Window): organização de informações em janelas sobrepostas;

Durante o desenvolvimento, foram trabalhados conceitos importantes de JavaScript:

Manipulação do DOM;
Eventos (addEventListener);
Estruturação e reutilização de código;
Organização modular;
Boas práticas de legibilidade e separação de responsabilidades;


HTML5;
CSS3;
JavaScript (ES6+);

O principal objetivo deste projeto foi consolidar o uso do JavaScript na criação de comportamentos interativos em páginas web, reforçando a importância da experiência do usuário (UX) e da organização do código.

Esse projeto foi essencial para:

Entender como tornar páginas estáticas em experiências dinâmicas;
Praticar lógica de programação aplicada ao front-end;
Melhorar a estruturação de código em projetos reais
